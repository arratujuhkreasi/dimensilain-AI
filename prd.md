PRD: DIMENSI LAIN SCRIPT PRODUCTION – Professional Horror Script Architect1. Ringkasan EksekutifDIMENSI LAIN SCRIPT PRODUCTION adalah platform berbasis web yang dirancang khusus untuk tim produksi "Dimensi Lain". Tujuannya adalah mengotomatisasi penulisan naskah horor profesional menggunakan Agentic AI yang ditenagai oleh Cerebras Inference. Sistem ini tidak hanya "sekali jadi", melainkan melalui proses wawancara (input form) untuk memastikan nuansa horor yang dihasilkan sesuai dengan visi sutradara.2. Target Pengguna & TujuanPengguna: Director of Production & Scriptwriter "Dimensi Lain".Masalah: Hambatan kreatif (writer's block) dan kesulitan menjaga konsistensi naskah untuk durasi panjang.Solusi: AI Agent yang bertindak sebagai Co-writer yang memahami struktur beat-sheet film horor.3. Fitur Utama (Functional Requirements)A. Intelligent Onboarding Form (Discovery Phase)Sebelum menulis, AI akan mengajukan pertanyaan krusial untuk menentukan "DNA" film:Judul & Logline: Ide dasar cerita.Sub-Genre: (Gore, Supernatural, Psychological, Found Footage, dll).Talent & Karakter: Nama, deskripsi fisik, dan kelemahan karakter (penting untuk horor).Setting/Lokasi: Memaksimalkan aset lokasi yang dimiliki YouTube "Dimensi Lain".Durasi: Pilihan 15, 30, 60, hingga 90 menit (menentukan jumlah Act).Jump Scare Density: Level ketakutan (Low, Medium, Extreme).B. Agentic Scriptwriting WorkflowSistem tidak menggunakan satu prompt besar, melainkan rantai agen:Agent 1 (Architect): Membuat struktur 3-Act atau 5-Act berdasarkan durasi.Agent 2 (Researcher): Menambahkan elemen mitos/legenda lokal agar terasa nyata.Agent 3 (Dialogue Master): Menulis percakapan agar tidak kaku.Agent 4 (Format Specialist): Memastikan output sesuai standar industri (Standard Screenplay Format).C. Export & PreviewPreview naskah dengan format ala naskah film (Font Courier Prime).Export ke PDF atau format .fdx (Final Draft).4. Tech Stack Rekomendasi (Modern & Fast)KomponenTeknologiAlasanFrontendNext.js 14+ (App Router)Performa cepat, SEO friendly, dan standar industri saat ini.UI LibraryShadcn UI + Tailwind CSSDesain modern, bersih, dan sangat mudah dikustomisasi (Dark Mode wajib).State ManagementZustandRingan untuk mengelola input form yang kompleks.AI EngineCerebras API (Llama 3.1 70B)Kecepatan inferensi hingga 1,000+ tokens/sec. Naskah 90 menit selesai dalam hitungan detik.OrchestrationLangGraph atau Simple Agentic LoopMengatur alur kerja antar agen AI secara sistematis.BackendFastAPI (Python)Sangat efisien untuk menangani logika AI dan streaming teks.5. Arsitektur Agen (The "Perfect" Logic)Untuk naskah berdurasi panjang (90 menit), AI tidak boleh menulis langsung dari awal sampai akhir karena akan kehilangan fokus (halusinasi). Gunakan metode Hierarchical Generation:Step 1: User mengisi Form.Step 2 (Agent Plot): Menghasilkan Outline bab per bab (Act 1, 2, 3).Step 3 (Agent Scene): User menyetujui outline, lalu AI menulis detil per Scene.Step 4 (Refinement): Agen terakhir memeriksa konsistensi karakter (misal: "Apakah si A yang sudah mati di menit 20 tiba-tiba muncul lagi di menit 50?").6. Desain UI/UX (Modern & Cinematic)Tema: Deep Dark Mode (Latar belakang #0a0a0a) dengan aksen Blood Red atau Neon Amber.Typography: Menggunakan font Sans-serif modern (Inter/Geist) untuk dashboard, dan Courier Prime untuk area naskah.Experience: Gunakan Skeleton Loaders saat AI sedang "berpikir" dan Typewriter Effect saat naskah dihasilkan agar terasa interaktif.7. Langkah Pengembangan (Action Plan untuk AI Agent Anda)Jika Anda menggunakan AI agent seperti Cline atau OpenHands untuk membangun ini, berikan instruksi berikut:Fase 1: "Buatlah boilerplate Next.js dengan Tailwind dan Shadcn UI. Implementasikan form multi-step untuk input detail film horor."Fase 2: "Integrasikan API Cerebras menggunakan SDK resmi. Buat fungsi untuk mengirim prompt secara berantai (Outline -> Script)."Fase 3: "Buat komponen 'Screenplay Renderer' yang memformat teks biasa menjadi format naskah film profesional dengan margin yang benar."Fase 4: "Tambahkan fitur download PDF menggunakan pustaka react-pdf atau serupa."Tips Profesional: Mengingat ini untuk YouTube "Dimensi Lain", pastikan ada input khusus untuk "Production Constraints" (misal: "Hanya punya 2 lokasi" atau "Hanya punya 3 aktor"). Ini akan membuat naskah yang dihasilkan AI jauh lebih mungkin untuk diproduksi tanpa membuang banyak biaya.
